# PAÊBIRÚ: Project Brief

## Visão Geral

**Nome do Projeto**: PAÊBIRÚ: A Jornada dos 21 Caminhos  
**Subtítulo**: Um Mapa Vivo para a Reconexão do Ser  
**Idioma**: Português brasileiro  
**Tipo**: Livro filosófico-espiritual

## Conceito Central

O Paêbirú é um **sistema de 21 arquétipos** organizados em 3 atos que funcionam como um mapa para o autoconhecimento e desenvolvimento pessoal. O conceito central é que **o caminhante já é o caminho** — não há destino a alcançar, apenas reconhecimento do que já se é.

### O 22º (O Paêbirú)
- Não é um arquétipo adicional, mas a **relacionalidade pura** entre os 21
- É "o que escreve", "o que mapeia", "o que caminha"
- Representa o vazio produtivo entre as manifestações
- Metáfora: "Os 21 são as notas. O 22º é a música."

## Estrutura de Cada Arquétipo

Cada capítulo de arquétipo segue esta estrutura:

1. **A Chegada** — Descrição sensorial do espaço sagrado
2. **A Aporia** — A tensão dialética irredutível (não pode ser "resolvida", apenas habitada)
3. **O Ensinamento** — Sabedoria profunda do arquétipo
4. **A Sombra** — O modo patológico quando a aporia é mal habitada
5. **A Sabedoria** — Síntese concisa
6. **O Portal** — Prática para encarnar o arquétipo
7. **A Ponte** — Transição para o próximo arquétipo

"Ditadura da Autenticidade" — ensaio sobre como o Paêbirú pode funcionar como protocolo social anti-manipulação. Conceitos:
- Anatomia do picareta (manipulador)
- Cibernética da autenticidade
- Estrutura anti-sombra

## Objetivos do Projeto

1. **Individual**: Oferecer mapa para autoconhecimento através de arquétipos habitáveis
2. **Relacional**: Expandir para tessitura com outros
3. **Coletivo**: Criar base para sistemas sociais anti-manipulação
4. **Filosófico**: Propor nova ontologia relacional (não há entidades que depois se relacionam; há relação da qual emergem entidades)

## Tom e Estilo

- **Filosófico** mas acessível
- **Poético** sem ser hermético
- **Prático** com exercícios concretos (Portais)
- **Não-dogmático**: apresenta aporias, não soluções
- **Inclusivo**: usa linguagem que ressoa com múltiplas tradições sem se filiar a nenhuma
