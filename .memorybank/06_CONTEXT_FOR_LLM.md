# Contexto Rápido para Modelos de Linguagem

## TL;DR do Projeto

**O que é**: Livro filosófico-espiritual com 21 arquétipos organizados em 3 atos, mais um 22º elemento que é a "relacionalidade pura" entre eles.

**Premissa central**: "Você já é o caminho" — não há destino a alcançar, apenas reconhecimento.

**Estrutura**:
- Ato I (1-7): Eu comigo — soberania interior
- Ato II (8-14): Eu com outros — tessitura relacional  
- Ato III (15-21): Além do eu — transcendência
- 22º: Não é arquétipo; é o espaço entre eles

**Tom**: Filosófico mas acessível, poético mas claro, paradoxal mas não confuso.

---

## Antes de Fazer Qualquer Edição

1. **Leia o arquivo relevante** — o memory bank é resumo, não substituto
2. **Verifique a estrutura padrão** — cada arquétipo segue formato específico
3. **Mantenha o tom** — convite, não imposição; aporia, não solução
4. **Preserve as metáforas espaciais** — cada arquétipo tem seu espaço sagrado

## Checklist de Qualidade

Ao editar qualquer capítulo de arquétipo, verificar:

- [ ] Tem todas as seções? (Chegada, Aporia, Ensinamento, Sombra, Sabedoria, Portal, Ponte)
- [ ] A aporia é tensão genuína (sem "solução fácil")?
- [ ] A sombra é distorção do arquétipo (não seu oposto)?
- [ ] O Portal é prático e executável?
- [ ] A Ponte conecta naturalmente ao próximo?
- [ ] O tom é convite, não imposição?
- [ ] Usa segunda pessoa ("você")?

## Erros Comuns a Evitar

| ❌ Evitar | ✅ Preferir |
|-----------|------------|
| "Você deve..." | "Você pode..." |
| "A solução é..." | "A aporia convida a..." |
| "O Guerreiro é superior ao..." | "O Guerreiro manifesta..." |
| "Supere a sombra" | "Reconheça a sombra" |
| "21 etapas para..." | "21 faces do caminhar" |

---

## Vocabulário Essencial

- **Paêbirú**: O caminho; a relacionalidade pura; o 22º
- **Aporia**: Tensão irredutível (não é problema a resolver)
- **Sombra**: Distorção patológica de um arquétipo
- **Portal**: Prática para encarnar o arquétipo
- **Ponte**: Transição entre arquétipos
- **Habitar**: Viver com/dentro de (vs. resolver/superar)
- **Manifestar**: O Paêbirú se expressando através de forma específica
